## 解法一、迴圈

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 336KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    while(cin >> n){
        int ans = 0;
        while(n > 0){
            n /= 2;
            ans++;
        }
        cout << ans << '\n';
    }
    
    return 0;
}
```

## 解法二、數學
