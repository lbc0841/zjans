## 解法一

如果 `a == b` 表示不需要翻轉<br>
如果 `a` 在 `b` 的背面表示需要翻轉 2 次<br>
如果都不是，則不管是往哪個方向翻轉都只需要 1 次

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (0.2s, 340KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int q;
    int a, b, vx1, vx2, vy1, vy2, vz1, vz2;
    
    cin >> q;
    for(int i=0; i<q; i++){
        cin >> a >> b >> vx1 >> vx2 >> vy1 >> vy2 >> vz1 >> vz2;
        
        if(a == b) cout << "0\n";
        else if(((a==vx1 && b==vx2) || (a==vx2 && b==vx1)) ||
                ((a==vy1 && b==vy2) || (a==vy2 && b==vy1)) ||
                ((a==vz1 && b==vz2) || (a==vz2 && b==vz1))) cout << "2\n";
        else cout << "1\n";
    }
    
    return 0;
}
```
