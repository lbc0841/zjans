## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 328KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    cin >> n;

    if(n >= 40) cout << 100;
    else if(n > 20) cout << 80 + (n-20);
    else if(n > 10) cout << 60 + 2*(n-10);
    else cout << n*6;

    return 0;
}
```
