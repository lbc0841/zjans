## 解法一、stringstream

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 344KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string line;
    while(getline(cin, line)){
        stringstream ss(line);
        int num, sum = 0;
        while(ss >> num) sum += num;
        
        cout << sum << "\n";
    }
    
    return 0;
}
```
