> 給一行加密後的字串 s<br>
加密方式為對每個字元的 ASCII 碼加上一個整數 k<br>
對其解碼並輸出

***

## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 352KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int k= '*'-'1';

    string s;
    cin >> s;

    for(char c : s){
        cout << (char)(c+k);
    } 

    return 0;
}
```
