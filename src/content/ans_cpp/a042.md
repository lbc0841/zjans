> 給一個數字 n<br>
求平面上的 n 個圓最多可將平面切成幾個區域

***

## 解法一、數學

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 348KB)**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    while(cin >> n) cout << (n-1)*n+2 << "\n";

    return 0;
}
```
