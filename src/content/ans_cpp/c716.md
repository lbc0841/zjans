> 給一個名字 s<br>
輸出 "Go, {s}, go go"

<font color="#ff2020">※ 名字可能含空格</font>

***

## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 328KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s;
    getline(cin, s);
    cout << "Go, " << s <<", go go";
    
    return 0;
}
```
