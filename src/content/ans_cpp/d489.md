> 給三個數 a, b, c，代表三角形土地的三邊長<br>
求土地面積的平方為多少

***

## 解法一

### 🔹 海龍公式

三角形面積 $A = \sqrt{s(s-a)(s-b)(s-c)}$<br>
其中 $s = \frac{a+b+c}{2}$，為三角形的半周長

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 348KB)**

```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int a, b, c, s;
    cin >> a >> b >> c;
    s = (a+b+c)/2;
    cout << s*(s-a)*(s-b)*(s-c);
    
    return 0;
}
```
