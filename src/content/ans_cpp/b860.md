### # 題目簡述

***

## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (4ms, 336KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int c, w;
    cin >> c >> w;
    
    int ans = 0;
    while(w>0){
        if(c >= 12){
            c -= 10;
            w--;
            ans++;
        }
        else{
            w -= (12-c);
            c = 12;
        }
    }
    cout << ans;
    
    return 0;
}
```
