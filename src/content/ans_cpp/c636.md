## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (2ms, 320KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s[] = {"鼠", "牛", "虎", "兔", "龍", "蛇",
                  "馬", "羊", "猴", "雞", "狗", "豬"};
    
    int y;
    while(cin >> y){
        if(y < 0) cout << s[(y+120)%12] << "\n";
        else cout << s[(y-1)%12] << "\n";
    }
    
    return 0;
}
```
