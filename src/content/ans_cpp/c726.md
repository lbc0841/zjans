> 給兩個由空白隔開的名字 s1, s2<br>
輸出 "{s1} and {s2} sitting in the tree"

***

## 解法一

### ✅ 完整代碼

評分結果(參考) ： **<font color="#00bb00">AC</font> (3ms, 336KB)**

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    string s1, s2;
    cin >> s1 >> s2;
    cout << s1 << " and " << s2 << " sitting in the tree";
    
    return 0;
}
```
